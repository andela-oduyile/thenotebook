# THE NOTEBOOK

A simple note taking app

## Features
- Create a Note
- View all Notes created
- Edit a given Note
- Search for a Note

## Technologies

- Ruby on Rails
- Materialize CSS

## Local Development

- Install npm dependencies bundle install
- To run the app: rails s
